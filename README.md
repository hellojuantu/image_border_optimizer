- [GenOptimizer 在线演示地址](https://genoptimizer.cn/?hidpi=true)

## 项目介绍
- 支持多图操作
- 支持图片拖拽添加
- 支持所有属性的动态配置
- 支持一键复制修改后的结果
- 支持画笔、文字、矩形、圆形、箭头、线条、图像的添加

这个项目没有依赖于前端框架, 以纯 JS 实现

最后抽象出了一个框架 ([GenOptimizer](doc/gen_optimizer.md)), 以一种十分简洁易用的方式写出了整个项目
